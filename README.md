# GitHub Users Data Analysis Project

## About This Project
This is a simple data analysis project where I used real data from the GitHub API. The goal was to understand GitHub users and explore basic patterns in the data.

## What I Did
- Collected data from GitHub API
- Converted JSON data into a pandas DataFrame
- Cleaned and selected useful columns
- Created a new feature: login length
- Performed basic data analysis (EDA)
- Checked user types and patterns
- Made visualizations like bar chart, pie chart, and histogram

## Key Insights
- Most accounts are normal users, very few are organizations
- Username lengths are mostly short
- There is no strong relationship between user ID and username length

## Tools Used
- Python
- Pandas
- Matplotlib
- Requests

## Dataset
GitHub API: https://api.github.com/users
